### Obter informações de saldo
{{EXAMPLE_QUERY}}

Este método permite obter informações sobre o saldo do usuário.

#### Resposta do servidor
Array de dados

Campo     | Tipo     | Descrição
---------|---------|-------------
balance  | fixed   | Valor atual do saldo do usuário com precisão de 4 casas decimais.
currency | char(3) | Moeda do saldo no formato [ISO 4217 alphabetic code](https://en.wikipedia.org/wiki/ISO_4217#cite_ref-exponent_7-0).
